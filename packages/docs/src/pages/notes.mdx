styled vs sx prop

## Styled

- props cleaning
- styled system props
- as prop
- forward refs

- Make this Reflexbox
  - Add sx prop support to components

## SX Prop

- sx and css prop
- smaller bundle
- less ergonomic (debatable)
- no base component
- potential for generative components

## What Knobs do people want

- per-instance overrides
- easy theming
- compose pieces together
- out-of-the-box productivity
- color + typography
- noticeable components
  - buttons
  - forms

## Theme vs Component

- code splitting and tree shaking
